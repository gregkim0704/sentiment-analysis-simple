# 센티멘트 분석 플랫폼 (간단 버전)

## 🎯 프로젝트 개요
멀티 스테이크홀더 센티멘트 분석 시스템의 간단한 데모 버전입니다.

## 🚀 기술 스택
- **프론트엔드**: React + JavaScript
- **배포**: Railway
- **스타일링**: 인라인 CSS

## 📦 설치 및 실행

### 로컬 개발
```bash
npm install
npm start
```

### 프로덕션 빌드
```bash
npm run build
```

## 🌐 배포
Railway를 통해 자동 배포됩니다.
